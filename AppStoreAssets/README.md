# App Store Assets

Upload-ready asset package for Snack Swap Adventure.

## Screenshots

Apple accepts 1 to 10 screenshots per display size. These sets use the current highest-priority portrait sizes:

- `Screenshots/iPhone-6.9/en-US/` — 1320 x 2868 PNG
- `Screenshots/iPad-13/en-US/` — 2048 x 2732 PNG

Suggested upload order:

1. `01-gameplay.png`
2. `02-world-map.png`
3. `03-time-up.png`
4. `04-result.png`
5. `05-title.png`

## Icon

- `Icon/AppIcon-1024.png` — 1024 x 1024 RGB PNG, flattened/no alpha
